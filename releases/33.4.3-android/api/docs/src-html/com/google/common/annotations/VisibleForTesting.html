017/**
018 * Annotates a program element that exists, or is more widely visible than otherwise necessary, only
019 * for use in test code.
020 *
021 * <p><b>Do not use this interface</b> for public or protected declarations: it is a fig leaf for
022 * bad design, and it does not prevent anyone from using the declaration---and experience has shown
023 * that they will. If the method breaks the encapsulation of its class, then its internal
024 * representation will be hard to change. Instead, use <a
025 * href="http://errorprone.info/bugpattern/RestrictedApi">RestrictedApiChecker</a>, which enforces
026 * fine-grained visibility policies.
027 *
028 * @author Johannes Henkel
029 */
030@GwtCompatible
031public @interface VisibleForTesting {
032}
